Manuals, user guides and more for in:control devices and software.

## Users

If you landed here by chance, welcome! Please use the following links.

Website: [in-control.ch](https://in-control.ch) 
Documentation: [docs.in-control.ch](https://docs.in-control.ch) *not yet released* 
Configurator: [configure.in-control.ch](https://configure.in-control.ch)

## Authors and Contributors

All documentation files are located in the `src` folder.
Images and other assets are located in the `assets` folder.

We use obsidian for local editing of the documentation but feel free to use any text editor of your choice.
Contribute by opening issues or pull requests.

Currently every push to the `main` branch is automatically deployed to [GH Pages](https://in-control-ch.github.io/documentation/) until we reach a first release.
